# Urmi Shield - Android App

This is the Android application module for Urmi Shield.

## Structure

-   `src/main/java`: Source code.
-   `src/main/res`: Resources (XML, images).
-   `src/main/assets`: ML models.

## Dependencies

-   **Compose**: UI framework.
-   **Room**: Local database.
-   **DataStore**: Preferences.
-   **WorkManager**: Background tasks.
-   **TensorFlow Lite**: AI inference.
-   **SQLCipher**: Database encryption.

## Build

Run `./gradlew assembleDebug` to build the APK.
